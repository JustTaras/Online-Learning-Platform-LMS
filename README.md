# Платформа LMS

Сучасна система управління навчанням (Learning Management System), розроблена на C# 11 з дотриманням принципів чистої архітектури та SOLID.

## Структура рішення

### Проекти

1. **LmsPlatform.Domain** - Шар доменної логіки
   - Абстрактний базовий клас `Entity` з унікальним ідентифікатором
   - Інтерфейс `IValidatable` для валідованих сутностей
   - Сутності: `Course`, `Lesson`, `Test`, `StudentProgress`
   - Строга інкапсуляція з валідацією в конструкторах та сеттерах
   - Перевантаження оператора `+` для додавання уроків до курсу

2. **LmsPlatform.App** - Застосунок для демонстрації
   - Приклади використання доменних сутностей
   - Базова точка входу в систему

3. **LmsPlatform.Tests** - Набір unit тестів
   - Тести для курсів, уроків та прогресу студентів
   - Валідація коректності бізнес-логіки
   - 8 успішних тестів

## Ключові особливості

### Entity базовий клас
- Автоматичне генерування `Guid` ID
- Можливість встановлення власного ID

### IValidatable інтерфейс
- Метод `bool Validate()` для перевірки коректності сутності

### Course сутність
- Назва та опис курсу
- Список уроків (тільки для читання через IReadOnlyList)
- Перевантаження оператора `+` для додавання уроків:
  ```csharp
  var course = new Course("C#", "Основи");
  var lesson = new Lesson("Вступ", "Опис", 1);
  course += lesson;
  ```

### Lesson сутність
- Назва, опис та порядковий номер
- Валідація довжини назви (макс. 255 символів)

### Test сутність
- Назва, опис та кількість запитань
- Валідація параметрів

### StudentProgress сутність
- Відстеження прогресу студента по курсу
- Автоматичний розрахунок відсотку завершення
- Методи для оновлення прогресу

## Компіляція та запуск

### Побудова рішення
```bash
dotnet build LmsPlatform.slnx
```

### Запуск тестів
```bash
dotnet test LmsPlatform.Tests/LmsPlatform.Tests.csproj
```

### Запуск застосунку
```bash
dotnet run --project LmsPlatform.App/LmsPlatform.App.csproj
```

## Архітектурні принципи

- **Інкапсуляція**: Приватні поля з публічними властивостями та валідацією
- **Валідація**: Всі сутності реалізують IValidatable
- **Розділення відповідальності**: Кожен клас має одну відповідальність
- **Типобезпека**: Використання Guid для ID замість простих чисел
- **Незмінюваність**: Список уроків повертається як IReadOnlyList
